A simple React app built using Google Maps for React and NASA open API.

This app uses the NASA Open API to gather event data for hurricanes, wildfires, volcanic events and icebergs and plot them onto a map. Each event has its own component with specific icons imported from react-icons.

On startup the app displays a loading spinner while it fetches the data, the data is as real time as the api allows. the map is then loaded.

Despite movement of icebergs only the most recent position is plotted, this is due to there being a surprising amount and the zoom level on the map which gives a clear overview is confusing if we plot all the recent locations. It just looks like a blurred line of icons around Antarctica

Volcanic events are plotted as single points.
Wildfires are plotted as single points.
Severe Storms are plotted as a chain of events.

Improvements:

  • Add Earthquake data
  • Add a shockwave radius to earthquake data
  • Add satellite imagery overlay to display actual storm sizes visually
  • Add an option for further info on the event to be linked in the Info Box
  • A better improvement would be a further Info component which bring in information on the event
  • Storm icons need to fade out as they get older
  • Storm icons should be different colors to represent strength
  • Storm icons could be linked with a path and also surrounded by an area affected
